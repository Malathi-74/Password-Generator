# Random Password Generator

## Description
This is a simple web application that generates random passwords.  
Users can choose the password length and generate a secure password instantly.

## Features
- Generate random password
- Custom password length
- Simple and clean UI

## Tech Used
- HTML
- CSS
- JavaScript (Internal Script)

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any browser.
3. Click generate to create a password.
